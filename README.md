# 🚀 v1.0.0 — Initial Release

Speech Coach의 첫 App Store 배포 버전입니다.  
말하기 분석과 피드백을 중심으로 한 MVP를 출시했습니다.

## Features
- 발화 속도 및 흐름 기반 말하기 분석
- 하이라이트 기반 코칭 피드백 제공
- 분석 기록 저장 및 재확인

## Improvements
- 분석/재생 흐름 안정화
- 홈 및 결과 화면 UX 개선
- 코칭 시트 UI 정리

## Fixes
- 영상 재생 관련 버그 수정
- 분석 결과 로딩 안정화

## Notes
- 본 버전은 MVP이며, 이후 기능 확장을 예정하고 있습니다.
